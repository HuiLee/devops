# top

top命令可以实时动态地查看系统的整体运行情况

## 参数

```
-b：以批处理模式操作； 
-c：显示完整的治命令； 
-d：屏幕刷新间隔时间； 
-I：忽略失效过程； 
-s：保密模式； 
-S：累积模式； 
-i<时间>：设置间隔时间； 
-u<用户名>：指定用户名； 
-p<进程号>：指定进程； 
-n<次数>：循环显示的次数;
```
